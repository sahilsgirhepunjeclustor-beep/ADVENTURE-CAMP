# **App Name**: TrailWise

## Core Features:

- Tri-Role Dashboard Architecture: Unique landing environments for Admins, Organizers, and Camp Explorers based on verified authentication status.
- Advanced Adventure Marketplace: High-fidelity listing interface with filtering by terrain type, group capacity, difficulty, and budget constraints.
- AI Dynamic Itinerary Tool: Leverages reasoning to generate custom 5-day trip plans and safety protocols based on user location and terrain type.
- Document Verification Pipeline: Dedicated system for organizers to upload business registrations and safety certifications for administrative review.
- Local Session Engine: Comprehensive use of LocalStorage to persist user settings, bookings, and camp inventories without a server-side database.
- Interactive Booking Workflow: End-to-end reservation system including live availability calendars, guest counters, and total price calculation.
- Visual Analytics Hub: SVG-based reporting module for tracking revenue trends, booking occupancy, and user growth across the platform.

## Style Guidelines:

- Primary Color: Forest Green (#169b56) – A deep, vibrant hue representing the core essence of wilderness adventure.
- Background: Desaturated Mint (#f0f7f2) – A clean, light HSL-derived shade to ensure high readability in editorial contexts.
- Accent: Deep Leaf (#128045) – An analogous darker hue providing 30-degree contrast for calls to action.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, tech-forward scientific aesthetic. Body Font: 'Inter' (sans-serif) for professional, machine-precision legibility.
- Modern sidebar architecture (260px) utilizing glassmorphism and sticky topbar with integrated live-clock and role indicators.
- Custom duo-tone icons for camp categories like 'Forest', 'Mountain', and 'Desert' to ensure thematic visual unity.
- Subtle slide-up entrance animations for auth cards and micro-interactions for gallery hover states.
